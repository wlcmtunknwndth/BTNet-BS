# BTNet: Нейронная сеть на основе биномиального дерева для оценки опционов

Реализация нейросетевых архитектур для оценки европейских и американских опционов пут на основе биномиальной модели Кокса-Росса-Рубинштейна (CRR).

## Описание проекта

Данный проект реализует подход, описанный в статье:
> Шорохов С.Г. Оценка финансовых деривативов нейронной сетью на основе биномиального дерева // International Journal of Open Information Technologies. – 2024. – Т. 12, № 5. – С. 108–114.

### Ключевая идея

Архитектура нейронной сети напрямую выводится из структуры биномиального дерева:
- **Обратный ход по дереву** → последовательность слоёв сети
- **Дисконтирование ожидаемой стоимости** → свёрточный слой с фильтром размера 2
- **Выбор между удержанием и досрочным исполнением** → maxout-слой

## Архитектуры

### BTNetEuropean — для европейских опционов

```
Вход (K) → DenseLayer (ReLU, n+1 нейронов) → ConvLayer × n → Выход
```

- Полносвязный слой с ReLU моделирует терминальные условия
- Последовательность свёрточных слоёв реализует дисконтирование

### BTNetAmerican — для американских опционов

```
Вход (K) → DenseLayer (ReLU, n+1 нейронов) → MaxoutLayer × n → Выход
```

- Maxout-слои реализуют выбор: `max(ожидаемая стоимость, выгода от исполнения)`
- Позволяет моделировать возможность досрочного исполнения

## Использование

### Пример: Оценка европейского опциона пут

```python
import torch
import numpy as np

# Параметры модели
S0 = 0.5      # Текущая цена актива
T = 1.0       # Срок до погашения
r = 0.05      # Безрисковая ставка
sig = 0.25    # Волатильность
n_dim = 9     # Число периодов

# Создание модели
model = BTNetEuropean(n_dim, S0, sig, T, t0=0.0, r=r)

# Обучение на данных Блэка-Шоулза
K_train = np.random.uniform(0.25, 0.75, 500).reshape(-1, 1)
prices_train = bs_put_price(S0, K_train, T, r, sig).reshape(-1, 1)

loss_history = train_BTNet(model, K_train, prices_train, epochs=200)

# Предсказание
K_test = np.array([0.4, 0.5, 0.6])
predictions = model.predict(K_test)
```

### Пример: Оценка американского опциона пут

```python
# Создание модели для американского опциона
model_american = BTNetAmerican(n_dim, S0, sig, T, t0=0.0, r=r)

prices_american = np.array([
```

## Математическая основа

### Биномиальная модель CRR

Множители движения цены:
```
u = exp(σ√Δt)    # рост
d = exp(-σ√Δt)   # падение
```

Риск-нейтральные вероятности:
```
πᵤ = (e^(rΔt) - d) / (u - d)
π_d = 1 - πᵤ
```

### Обратный ход для европейского опциона

```
V(i,j) = e^(-rΔt) · [πᵤ · V(i+1,j+1) + π_d · V(i+1,j)]
```

### Обратный ход для американского опциона пут

```
V(i,j) = max{e^(-rΔt) · [πᵤ · V(i+1,j+1) + π_d · V(i+1,j)], K - S(i,j)}
```
