**Постановка задачи**
Оценка американского пут-опциона — это задача оптимальной остановки с подвижной границей, для которой нет аналитического решения. Стандартный подход — биномиальное дерево CRR — решает её обратной индукцией. В работе моего научного руководителя — Шорохова Сергея Геннадьевича — показано, что прямой проход нейронной сети специальной архитектуры математически эквивалентен этой индукции. В моей ВКР я реализую и верифицирую такую архитектуру для оценки американских пут-опционов, сравниваю результаты с эталонными методами и исследую возможность вычисления греческих символов опционов через автоматическое дифференцирование для последующей оценки рыночного риска.

**Основные результаты**
1. Реализован пакет btnn_bs для оценки американских опционов при помощи нейронных сетей с архитектурами BTNetEuropean и BTNetAmerican.
2. Аналитическая инициализация весов из параметров CRR подтверждает теоретическую эквивалентность: MAE = 2.84·10^-4 относительно QuantLib CRR (500 шагов).
3. Вычислены греческие символы опционов (Delta, Gamma, Vega, Theta) через torch autograd; выявлен эффект Gamma = 0 как фундаментальное ограничение ReLU-архитектур.
4. В эксперименте с переносом весов показано, что отклонение весового фильтра на dW ≈ 0.004 приводит к ошибке Delta MAE ~ 8·10^-3 — греческие символы опционов чувствительны к точности инициализации.
