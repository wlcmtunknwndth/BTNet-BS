# Бриф встречи с научным руководителем — 17 апреля 2025

## Текущие задачи

- Доделать вычисление греческих символов для модели с переносом весов (transfer learning)
- Объяснить практическую применимость греческих символов: если умеем их вычислять, можем использовать для оценки рыночного риска

## Ключевая идея ВКР

Задача оценки американского пут-опциона — это задача с подвижной краевой границей, для которой нет аналитического решения. Наш подход — обходной манёвр: нейросетевая архитектура BTNet, прямой проход которой математически эквивалентен обратной индукции по биномиальному дереву CRR.

## Структура ВКР

1. Гиперпараметры модели

3. Вычислительные эксперименты
4. Исследование греческих символов опционов для последующей оценки рисков
